# FajniejszeAK

Za zaliczenie NASZE i WASZE

Możesz poruszać się między pytaniami, używając przycisków na ekranie lub fizycznych klawiszy na klawiaturze.

P - prawda
F - fałsz
Aby przejść do następnego pytania, kliknij dowolny przycisk.
